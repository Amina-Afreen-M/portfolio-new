# Projects Directory

This directory contains project-related assets and thumbnails.

Organize your project assets in the following structure:

```
projects/
├── project1/
│   ├── thumbnail.jpg
│   └── other-assets/
├── project2/
│   ├── thumbnail.jpg
│   └── other-assets/
└── ...
```

Ensure all images are optimized for web use to maintain fast loading times.
